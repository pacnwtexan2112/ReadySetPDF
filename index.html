API & CLI — get a PDF in one request

Example: POST HTML and get a PDF back.

// curl
curl -X POST "https://api.readysetpdf.com/v1/render" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "html": "
Hello world
", "options": { "format": "A4" } }' \
  --output report.pdf

// Node (example)
import fetch from "node-fetch";
const res = await fetch("https://api.readysetpdf.com/v1/render", {
  method: "POST",
  headers: { "Authorization": "Bearer X", "Content-Type": "application/json" },
  body: JSON.stringify({ html: "
Hi
" })
});
const buffer = Buffer.from(await res.arrayBuffer());
await fs.promises.writeFile("out.pdf", buffer);

Options & templates

Pass headers and footers, page numbers, custom CSS, and choose page size/orientation. Templates let you standardize branding across generated docs.

Example options
  • format: "A4" or "Letter"
  • margins: { top: 20, bottom: 20 }
  • headerHtml, footerHtml
  • scale, displayHeaderFooter, preferCSSPageSize
